## Fox Game

## Description
This projects lets players control a fox that moves around and collects leafs. There is visual and audio feedback if the fox takes any damage from the boars or fire. There is also positive audio feedback when the fox successfully collects leaves.

---
## Interaction Instructions
Use WASD or arrow keys to move around and jump, and T to toggle debug menu. 
Use space key to attack enemies.
